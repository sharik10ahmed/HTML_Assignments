Future Roadmap

  • Frontend Technology Stack Migration
    Frontend: Next.js (React) or Vue.js (Nuxt.js) with TailwindCSS for styling and React Query for async state management.
    Backend: Keep FastAPI as a pure JSON REST API (it is already well-structured for this). Drop Jinja2 completely.
  • SMS Alerts & Paid Subscription Plans
    SMS Integration: Integrate an SMS gateway (e.g., Twilio, MSG91, or AWS SNS) directly into the existing Taskiq/NATS worker pool to send high-priority SMS alerts alongside emails.
    Subscription Tiering (SaaS Model): Expand the User or Profile models to include subscription_tier (Free, Pro, Enterprise). Use Razorpay Subscriptions (Recurring billing API) to gate premium features (e.g., higher profile visibility for lawyers).
